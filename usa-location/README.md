# 🛠️ 开发者工具箱

一个集成多种实用开发工具的现代化工具集合，为开发者和创作者提供高效便捷的解决方案。

## ✨ 工具列表

### 🟢 可用工具
- **🇺🇸 美国虚拟身份生成器**: 生成真实格式的美国地址和完整虚构个人信息
  - 真实地址格式生成
  - 完整个人档案信息
  - 州/城市精确筛选
  - 免税州筛选功能
  - 批量生成和导出
  - 地图可视化定位

- **💻 编程软件下载中心**: 汇聚常见编程开发工具的官方下载地址和安装教程
  - IDE/编辑器工具集合
  - 运行时环境下载
  - 数据库工具资源
  - 容器化工具链接
  - 官方教程和文档

- **🔐 双向文本加密解密**: 支持主流加密算法和古典密码的专业加密解密工具

  - 古典加密算法（凯撒密码、维吉尼亚密码、栅栏密码、摩斯密码等）
  - 双向加密解密功能
  - 左右分栏布局设计
  - 密钥管理和安全提示

## 🚀 快速开始

### 环境要求
- Node.js 18+
- npm 或 yarn

### 安装步骤

1. **安装依赖**
```bash
npm install
```

2. **启动开发服务器**
```bash
npm run dev
```

3. **访问应用**
打开浏览器访问 `http://localhost:3001`

## �📖 使用说明

### 工具箱主页
1. **浏览工具** - 在主页查看所有可用和即将推出的工具
2. **搜索工具** - 使用搜索栏快速查找需要的工具，支持按名称、功能、分类搜索
3. **选择工具** - 点击工具卡片进入对应的工具页面
4. **返回主页** - 在任何工具页面都可以通过导航返回主页

### 美国虚拟身份生成器
1. **选择筛选条件** - 可选择特定州或城市，支持免税州筛选
2. **设置生成数量** - 选择需要生成的身份档案数量（1-50个）
3. **一键生成** - 点击"生成身份"按钮
4. **查看结果** - 在页面下方查看生成的详细信息
5. **地图定位** - 在地图上查看生成地址的位置
6. **导出数据** - 选择JSON、CSV或TXT格式导出

### 编程软件下载中心
1. **浏览分类** - 按工具类型（IDE、运行时、数据库等）浏览
2. **搜索工具** - 使用搜索功能快速找到需要的开发工具
3. **查看详情** - 点击工具卡片查看详细信息和特性
4. **官方下载** - 直接跳转到官方下载页面
5. **查看教程** - 访问官方文档和安装教程

### 双向文本加密解密
1. **选择算法类型** - 左侧主流加密算法，右侧古典加密算法
2. **输入文本** - 在文本框中输入要加密或解密的内容
3. **设置密钥** - 对于需要密钥的算法，输入相应的密钥
4. **执行操作** - 选择加密或解密操作并执行
5. **查看结果** - 在结果区域查看处理后的文本
6. **复制结果** - 一键复制结果到剪贴板

## ⚠️ 重要声明

**本工具箱中的所有工具仅供学习、测试和开发使用。**

### 虚拟身份生成器
- 所有个人信息都是随机生成的虚假数据
- SSN、信用卡号等敏感信息均为无效格式
- 地址信息虽然格式真实，但不对应实际地址
- 请勿将生成的虚假信息用于任何非法用途或欺诈行为

### 加密解密工具
- 本工具仅供学习和测试使用，请勿用于加密敏感信息
- 古典加密算法安全性较低，不适用于现代安全需求
- 哈希算法（MD5、SHA-256）是单向的，无法解密
- 请妥善保管您的密钥，丢失密钥将无法解密数据
- 建议在生产环境中使用更安全的加密方案
